Finished the MIPS CPU

Apr 9, 2016 | Hits

Yesterday, I’ve finished my own MIPS CPU. This CPU:

  • Based on FPGA platform, use Nexys 4.
  • Used Verilog Hardware Describe Language.

It have these features:

  1. Have 27 instructions, include algorithm instructions, branch instructions, jump instructions and syscall instruction.
  2. 5 stage pipline. 5 stages:
    • IF(Instruction Fetch)
    • ID(Instruction decode)
    • EX(Execute)
    • MEM(Memory)
    • WB(Write back)
  3. Can insert bubbles for data dependence.
  4. Can support data bypass.

I’ve spent more than 70 hours on it. Verily/FPGA seems a little difficult for me. In my class, others used the Logisim to accomplish the same thing.

Here, you can find MIPS_CPU on GitHub, this repository have several branches.
